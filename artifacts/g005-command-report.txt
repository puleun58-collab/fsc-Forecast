G005 / G002 re-verification commands
=================================

1. npm run fetch:opinet
2. npm run fetch:opinet  # repeat run to verify same-date JSON update without duplication
3. npx tsc --noEmit
4. npx prisma validate
5. npx prisma generate
6. node --import tsx --input-type=module embedded-postgres verifier:
   - start embedded Postgres on 127.0.0.1:55436
   - prisma db push --skip-generate against the verifier DB
   - monkey-patch fetch so worker uses a deterministic local Opinet diesel payload from data/oil-price-daily.json
   - runScheduledWorkerOnce() twice
   - runIndicatorSync() twice with the same timestamps and changed values/source payloads
   - query ingest_runs / price_revision_log / daily_price_current / recompute_snapshots / external_indicator_history
   - assert latest current-truth row points at the latest recompute snapshot
